HTML Table Refference

Table Tags

Tag Name Description
<table> Table Creates the table
<thead> Table Head Defines the set of rows defining the headers of the column. Encapsulates <tr> and inside of that a <th>
<tbody> Table Body <tbody>element encapsulates a set of <tr>, indicating that they comprise the body of the table. Inside of the <tr> you can have a <th> and <td>.
<tr> Table Rows Creates the rows of whihch you put information in
<td> Table Data Contains data/information inside of the table. Normally after the Table Head <th> tag(s)
<tfoot> Table Footer <tfoot>element is like the header but for the bottom (footer). Can be used to show totals and such. Encapsulates <tr> inside of that <th> and <td>

Table Attributes

Attribute Name Description
Colspan Column Span Defines how many columns a td element spans.
Rowspan Row Span Defines how many rows a td element spans.
